# Enhanced Video Favorites

動画サイト用の高機能お気に入り管理ツール

## 概要

Enhanced Video Favoritesは、動画サイトでのお気に入り管理を効率化するブラウザ拡張機能です。動画のURLやタイトル、サムネイル画像を簡単に保存・管理できます。

## 主な機能

- **ワンクリック追加**: 現在閲覧中のページを簡単にお気に入りに追加
- **手動URL追加**: 任意のURLを手動で追加可能
- **カテゴリ管理**: お気に入りをカテゴリ別に整理
- **タグ機能**: 柔軟なタグ付けによる分類
- **サムネイル表示**: 動画のサムネイル画像を自動取得・表示
- **Web画面**: 専用のWeb画面でより詳細な管理が可能
- **コンテキストメニュー**: 右クリックメニューからの操作
- **通知機能**: 操作完了時の通知表示

## インストール方法

### 開発版のインストール

1. このリポジトリをクローンまたはダウンロード
```bash
git clone [リポジトリURL]
cd enhanced-video-favorites
```

2. ブラウザの拡張機能管理画面を開く
   - **Firefox**: `about:debugging` → 「この Firefox」→ 「一時的なアドオンを読み込む」
   - **Chrome**: `chrome://extensions/` → 「デベロッパーモード」を有効 → 「パッケージ化されていない拡張機能を読み込む」

3. プロジェクトフォルダを選択してインストール

## 使用方法

### 基本操作

1. **お気に入り追加**
   - ブラウザのツールバーにある拡張機能アイコンをクリック
   - 「現在のページを追加」ボタンでワンクリック追加
   - 「手動でURL追加」で任意のURLを追加

2. **お気に入り管理**
   - ポップアップの「一覧」タブで保存済みアイテムを確認
   - 「Web画面で開く」でより詳細な管理画面を表示

3. **カテゴリとタグ**
   - 「設定」タブでカテゴリとタグを管理
   - お気に入り追加時にカテゴリとタグを選択可能

### 右クリックメニュー

- ページ上で右クリック → 「お気に入りに追加」
- 画像上で右クリック → 「画像付きでお気に入りに追加」

## 対応ブラウザ

- Firefox (推奨)
- Chrome/Chromium系ブラウザ

## 権限について

この拡張機能は以下の権限を使用します：

- `activeTab`: 現在のタブ情報の取得
- `storage`: お気に入りデータの保存
- `tabs`: タブ操作
- `contextMenus`: 右クリックメニューの追加
- `notifications`: 通知の表示
